В языках Си, С++, Java, C#, JavaScript и других символ «&» применяется для обозначения нескольких операторов:

— для получения указателя переменной, унарный оператор, «&» должен предшествовать идентификатору (имени) переменной;

— оператор«&» означает побитовое И;

— оператор «&&» обозначает логическое И (проверка истинности последующего выражения только при условии истинности предшествующего).

Задание №6. Создайте блок статей

В Spotify появился режим караоке

Для него не требуется платная подписка.

Spotify анонсировал новую функцию Lyrics — по сути, режим караоке, который позволяет подпевать, даже если слова песни вспомнить не получается.

Для этого в мобильном приложении достаточно открыть окно воспроизведения и свайпнуть снизу вверх, чтобы на экране появился текст, который подсвечивает строчки по мере их звучания. В десктопном приложении функция включается нажатием на кнопку в виде микрофона, а в SpotifyTV можно включить или отключить Lyrics для всех воспроизводимых треков. Функция постепенно раскатывается для пользователей приложений для iOS, Android, ПК, игровых консолей и умных телевизоров. Она будет доступна владельцам как Premium-подписки, так и бесплатной версии. Если у вас она пока не появилась, придётся подождать несколько дней: апдейт глобальный и может занять некоторое время.
